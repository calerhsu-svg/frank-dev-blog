第1章：Unityのインストールと環境設定

Unityは世界で最も人気のあるゲームエンジンの一つで、2D/3Dゲーム開発、AR/VR、シミュレーションなど幅広い用途に対応しています。

1.1 Unity Hubのダウンロード

Unity公式サイト（unity.com）からUnity Hubをダウンロードします。Hubはバージョン管理とプロジェクト管理の統合ツールです。LTS（長期サポート版）の使用を推奨します。

💡 初心者はLTSバージョン（例：2022.3 LTS）を選びましょう。Tech Streamは不安定な新機能が含まれる場合があります。

1.2 最初のプロジェクト作成

プロジェクト設定の推奨：
├── プロジェクト名：MyFirstGame
├── テンプレート：2D (Built-in Render Pipeline)
├── 保存先：デスクトップ/UnityProjects/
└── Unityバージョン：2022.3 LTS

自動作成されるディレクトリ構造：
Assets/          ← すべてのゲームリソース
Library/         ← Unityキャッシュ（手動変更禁止）
Packages/        ← パッケージ管理
ProjectSettings/ ← プロジェクト設定

1.3 インターフェース概要

  • Scene：シーン編集ウィンドウ
  • Game：ゲームプレビューウィンドウ
  • Hierarchy：オブジェクト階層
  • Inspector：プロパティパネル
  • Project：アセット管理

第2章：インターフェースガイド

2.1 シーンナビゲーション

シーンビューの操作：
  マウス中ボタンドラッグ → パン
  スクロール           → ズーム
  Alt + 左ボタン       → 回転（3D）
  Fキー               → 選択オブジェクトにフォーカス

2.2 Transformツール

  • W：移動ツール（Move）
  • E：回転ツール（Rotate）
  • R：スケールツール（Scale）
  • T：矩形ツール（Rect）— UI要素用

2.3 Playモード

上部ツールバーのPlayボタン（またはCtrl+P）でゲームモードに入ります。

⚠️ 注意：Playモード中の変更は停止後にすべて消えます！必ず停止してから編集してください。

第3章：C#プログラミング基礎

3.1 最初のスクリプト

using UnityEngine;

public class PlayerController : MonoBehaviour
{
    // publicの変数はInspectorに表示されます
    public float speed = 5f;
    public float jumpForce = 10f;

    // Start は最初のフレームの前に一度だけ呼ばれます
    void Start()
    {
        Debug.Log("ゲーム開始！");
    }

    // Update は毎フレーム呼ばれます
    void Update()
    {
        float h = Input.GetAxis("Horizontal");
        transform.Translate(Vector3.right * h * speed * Time.deltaTime);

        if (Input.GetKeyDown(KeyCode.Space))
        {
            GetComponent<Rigidbody2D>().AddForce(Vector2.up * jumpForce, ForceMode2D.Impulse);
        }
    }
}

3.2 重要な概念

  • MonoBehaviour：すべてのUnityスクリプトの基底クラス
  • Start()：初期化、一度だけ実行
  • Update()：毎フレーム実行
  • Time.deltaTime：フレーム間隔時間

第4章：GameObjectとComponent

Unityの核心は「すべてはGameObject、機能はComponentで追加」という設計思想です。

4.1 よく使うComponent

  • Transform：位置・回転・スケール（必須）
  • SpriteRenderer：2D画像描画
  • Rigidbody2D：2D物理
  • Collider2D：2D当たり判定
  • AudioSource：音声再生
  • Animator：アニメーション制御

4.2 Prefab

// Prefabの生成
public GameObject bulletPrefab;

void Shoot()
{
    GameObject bullet = Instantiate(bulletPrefab, transform.position, Quaternion.identity);
    bullet.GetComponent<Rigidbody2D>().velocity = Vector2.right * 10f;
    Destroy(bullet, 3f);
}

第5章：物理システム入門

5.1 物理設定

キャラクター物理の設定：
1. PlayerのGameObjectを選択
2. Add Component → Rigidbody2D
   - Body Type: Dynamic
   - Gravity Scale: 1
   - Freeze Rotation Z: ✅
3. Add Component → BoxCollider2D

5.2 衝突判定

void OnCollisionEnter2D(Collision2D collision)
{
    if (collision.gameObject.CompareTag("Enemy"))
    {
        Debug.Log("敵に接触！");
    }
}

void OnTriggerEnter2D(Collider2D other)
{
    if (other.CompareTag("Coin"))
    {
        score += 10;
        Destroy(other.gameObject);
    }
}
💡 Collision＝物理衝突（跳ね返る）、Trigger＝トリガー（通過するが検知する）。コイン取得やエリア侵入にはTriggerを使います。

第6章：UIシステム

6.1 UI作成手順

1. Hierarchy右クリック → UI → Canvas
   - Canvas Scaler → Scale With Screen Size
   - Reference Resolution: 1920 x 1080
2. Canvas右クリック → UI → Text - TextMeshPro
3. Canvas右クリック → UI → Button - TextMeshPro

6.2 スクリプトでUI制御

using UnityEngine;
using TMPro;

public class UIManager : MonoBehaviour
{
    public TextMeshProUGUI scoreText;
    public GameObject gameOverPanel;
    private int score = 0;

    public void AddScore(int points)
    {
        score += points;
        scoreText.text = "Score: " + score;
    }

    public void ShowGameOver()
    {
        gameOverPanel.SetActive(true);
        Time.timeScale = 0f;
    }
}

第7章：アニメーションシステム

7.1 スプライトアニメーション作成

手順：
1. キャラクター選択 → Window → Animation → Animation
2. Create → "PlayerIdle.anim" と命名
3. 画像シーケンスをタイムラインにドラッグ
4. Samples（フレームレート）を調整：通常8-12 FPS

7.2 Animator Controller

Animator anim = GetComponent<Animator>();
anim.SetFloat("Speed", Mathf.Abs(h));
anim.SetBool("IsGrounded", isGrounded);
anim.SetTrigger("Jump");
anim.SetTrigger("Attack");

第8章：オーディオとパーティクル

8.1 音声再生

public class SoundManager : MonoBehaviour
{
    public static SoundManager Instance;
    public AudioClip jumpSound;
    public AudioClip bgm;
    private AudioSource audioSource;

    void Awake()
    {
        Instance = this;
        audioSource = GetComponent<AudioSource>();
        audioSource.clip = bgm;
        audioSource.loop = true;
        audioSource.Play();
    }

    public void PlaySFX(AudioClip clip)
    {
        audioSource.PlayOneShot(clip);
    }
}

8.2 パーティクルエフェクト

public GameObject explosionPrefab;

void OnDestroy()
{
    Instantiate(explosionPrefab, transform.position, Quaternion.identity);
}

第9章：シーン管理

using UnityEngine.SceneManagement;

SceneManager.LoadScene("GameScene");
SceneManager.LoadScene(SceneManager.GetActiveScene().buildIndex);

IEnumerator LoadAsync(string sceneName)
{
    AsyncOperation op = SceneManager.LoadSceneAsync(sceneName);
    while (!op.isDone)
    {
        float progress = Mathf.Clamp01(op.progress / 0.9f);
        loadingBar.value = progress;
        yield return null;
    }
}

第10章：ビルドと公開

File → Build Settings
1. すべてのシーンを追加（Scenes In Build）
2. プラットフォーム選択：
   - PC/Mac → Build And Run
   - Android → Android SDK必要
   - iOS → Mac + Xcode必要
3. Player Settings：
   - Company Name / Product Name
   - Icon / Resolution
   - Bundle Identifier
