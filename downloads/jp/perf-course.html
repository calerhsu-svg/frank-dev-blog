第1章：プロファイラー入門

最適化の第一歩は「推測ではなく計測」です。Unity ProfilerとCocosのDebugパネルを活用しましょう。

Unity Profiler: Window → Analysis → Profiler (Ctrl+7)

主要指標：
├── CPU Usage    → フレームごとのCPU消費
├── GPU Usage    → レンダリング消費
├── Memory       → メモリ割り当て
├── Rendering    → Draw Call、ポリゴン数
└── Physics      → 物理計算消費

目標値（60fps）：
  フレーム予算 = 1000ms / 60 = 16.67ms
  CPU < 12ms
  Draw Call < 100（モバイル）
  ポリゴン < 100K（モバイル）
💡 必ずターゲットデバイスでテストしてください。エディタの数値は実機と異なります。
