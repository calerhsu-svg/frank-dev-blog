Chapter 1: Installing Unity & Environment Setup

Unity is one of the world's most popular game engines, supporting 2D/3D game development, AR/VR, simulations, and more.

1.1 Download Unity Hub

Visit unity.com to download Unity Hub. It's the unified tool for managing Unity versions and projects. Always use the LTS (Long Term Support) version for stability.

💡 Beginners should choose the LTS version (e.g., 2022.3 LTS) instead of Tech Stream to avoid unstable features.

1.2 Create Your First Project

Recommended Project Settings:
├── Project Name: MyFirstGame
├── Template: 2D (Built-in Render Pipeline)
├── Location: Desktop/UnityProjects/
└── Unity Version: 2022.3 LTS

Auto-generated Directory Structure:
Assets/          ← All game resources
Library/         ← Unity cache (don't modify)
Packages/        ← Package management
ProjectSettings/ ← Project settings

1.3 Interface Overview

  • Scene: Scene editing window

Chapter 3: C# Programming Fundamentals

3.1 Your First Script

using UnityEngine;

public class PlayerController : MonoBehaviour
{
    // Public variables appear in the Inspector
    public float speed = 5f;
    public float jumpForce = 10f;

    // Called once before the first frame
    void Start()
    {
        Debug.Log("Game started!");
    }

    // Called every frame
    void Update()
    {
        float h = Input.GetAxis("Horizontal");
        transform.Translate(Vector3.right * h * speed * Time.deltaTime);

        if (Input.GetKeyDown(KeyCode.Space))
        {
            GetComponent<Rigidbody2D>().AddForce(Vector2.up * jumpForce, ForceMode2D.Impulse);
        }
    }
}

3.2 Key Concepts

  • MonoBehaviour: Base class for all Unity scripts
  • Start(): Initialization, runs once
  • Update(): Runs every frame
  • Time.deltaTime: Time between frames for consistent movement

Chapter 4: GameObject & Component System

Unity's core philosophy: everything is a GameObject, functionality comes from Components.

4.1 Common Components

  • Transform: Position, rotation, scale (required)
  • SpriteRenderer: 2D image rendering
  • Rigidbody2D: 2D physics
  • Collider2D: 2D collision detection
  • AudioSource: Audio playback
  • Animator: Animation control

4.2 Prefabs

public GameObject bulletPrefab;

void Shoot()
{
    GameObject bullet = Instantiate(bulletPrefab, transform.position, Quaternion.identity);
    bullet.GetComponent<Rigidbody2D>().velocity = Vector2.right * 10f;
    Destroy(bullet, 3f);
}

Chapter 5: Physics System Basics

5.1 Collision Detection

void OnCollisionEnter2D(Collision2D collision)
{
    if (collision.gameObject.CompareTag("Enemy"))
        Debug.Log("Hit an enemy!");
}

void OnTriggerEnter2D(Collider2D other)
{
    if (other.CompareTag("Coin"))
    {
        score += 10;
        Destroy(other.gameObject);
    }
}
💡 Collision = physical collision (bounces off); Trigger = passes through but detects contact. Use Triggers for pickups and area detection.

Chapter 6: UI System

using UnityEngine;
using TMPro;

public class UIManager : MonoBehaviour
{
    public TextMeshProUGUI scoreText;
    public GameObject gameOverPanel;
    private int score = 0;

    public void AddScore(int points)
    {
        score += points;
        scoreText.text = "Score: " + score;
    }

    public void ShowGameOver()
    {
        gameOverPanel.SetActive(true);
        Time.timeScale = 0f;
    }
}

Chapter 8: Audio & Particle Effects

public class SoundManager : MonoBehaviour
{
    public static SoundManager Instance;
    public AudioClip jumpSound;
    public AudioClip bgm;
    private AudioSource audioSource;

    void Awake()
    {
        Instance = this;
        audioSource = GetComponent<AudioSource>();
        audioSource.clip = bgm;
        audioSource.loop = true;
        audioSource.Play();
    }

    public void PlaySFX(AudioClip clip)
    {
        audioSource.PlayOneShot(clip);
    }
}
