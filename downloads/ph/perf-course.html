Chapter 1: Introduction to Profiling Tools

The first step in optimization is measuring, not guessing. Unity Profiler is your most important weapon.

Unity Profiler: Window → Analysis → Profiler (Ctrl+7)

Key Metrics:
├── CPU Usage    → Per-frame CPU time
├── GPU Usage    → Rendering time
├── Memory       → Memory allocations
├── Rendering    → Draw Calls, triangle count
└── Physics      → Physics calculation time

Target Values (60fps):
  Frame budget = 1000ms / 60 = 16.67ms
  CPU < 12ms
  Draw Calls < 100 (mobile)
  Triangles < 100K (mobile)
💡 Always test on target devices. Editor performance numbers don't represent real device performance.

Chapter 4: Physics Engine Tuning

// Layer Collision Matrix → Disable unnecessary collision layers
✅ BoxCollider2D, CircleCollider2D
❌ PolygonCollider2D (too many vertices)

Rigidbody2D.sleepMode = RigidbodySleepMode2D.StartAsleep;
💡 Layer Collision Matrix is the most overlooked optimization. Disabling unnecessary layers greatly reduces physics calculations.
