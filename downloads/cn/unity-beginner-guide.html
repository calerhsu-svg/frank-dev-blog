第一章：Unity 安装与环境设置

Unity 是目前全球最主流的游戏引擎之一，支持 2D/3D 游戏开发、AR/VR、模拟应用等多种场景。

1.1 下载 Unity Hub

前往 Unity 官网（unity.com）下载 Unity Hub。Hub 是管理 Unity 版本与项目的统一工具，建议使用 LTS 版本。

💡 建议：初学者请选 LTS 版本（如 2022.3 LTS），稳定性最高。

1.2 创建第一个项目

项目设置建议：
├── 项目名称：MyFirstGame
├── 模板：2D (Built-in Render Pipeline)
├── 位置：桌面/UnityProjects/
└── Unity 版本：2022.3 LTS

目录结构：
Assets/          ← 所有游戏资源
Library/         ← Unity 缓存（不要手动修改）
Packages/        ← 包管理
ProjectSettings/ ← 项目设置

1.3 界面初览

  • Scene：场景编辑窗口
  • Game：游戏预览窗口
  • Hierarchy：对象层级
  • Inspector：属性面板
  • Project：资源管理

第二章：界面导览与基本操作

2.1 场景导航

场景视窗操控：
  鼠标中键拖拽     → 平移视角
  滚轮             → 缩放
  Alt + 左键拖拽   → 旋转视角（3D）
  F 键             → 聚焦选中对象

2.2 Transform 工具

  • W：移动工具（Move）
  • E：旋转工具（Rotate）
  • R：缩放工具（Scale）
  • T：矩形工具（Rect）— UI 元素专用
⚠️ 注意：Play 模式下的任何修改在停止后都会消失！养成习惯先停止再修改。

第三章：C# 编程基础

3.1 第一个脚本

using UnityEngine;

public class PlayerController : MonoBehaviour
{
    public float speed = 5f;
    public float jumpForce = 10f;

    void Start()
    {
        Debug.Log("游戏开始！");
    }

    void Update()
    {
        float h = Input.GetAxis("Horizontal");
        transform.Translate(Vector3.right * h * speed * Time.deltaTime);

        if (Input.GetKeyDown(KeyCode.Space))
        {
            GetComponent<Rigidbody2D>().AddForce(Vector2.up * jumpForce, ForceMode2D.Impulse);
        }
    }
}

3.2 重要概念

  • MonoBehaviour：所有 Unity 脚本的基类
  • Start()：初始化，只运行一次
  • Update()：每帧执行
  • Time.deltaTime：帧间隔时间

3.3 常用 API

第四章：GameObject 与 Component

Unity 的核心理念是「一切皆 GameObject，功能靠 Component」。

4.1 常用 Component

  • Transform：位置、旋转、缩放（必须）
  • SpriteRenderer：2D 图片渲染
  • Rigidbody2D：2D 物理
  • Collider2D：2D 碰撞检测
  • AudioSource：音频播放
  • Animator：动画控制

4.2 Prefab 预制体

public GameObject bulletPrefab;

void Shoot()
{
    GameObject bullet = Instantiate(bulletPrefab, transform.position, Quaternion.identity);
    bullet.GetComponent<Rigidbody2D>().velocity = Vector2.right * 10f;
    Destroy(bullet, 3f);
}

第五章：物理系统入门

5.1 碰撞检测

void OnCollisionEnter2D(Collision2D collision)
{
    if (collision.gameObject.CompareTag("Enemy"))
        Debug.Log("碰到敌人！");
}

void OnTriggerEnter2D(Collider2D other)
{
    if (other.CompareTag("Coin"))
    {
        score += 10;
        Destroy(other.gameObject);
    }
}
💡 Collision = 物理碰撞（会弹开）；Trigger = 触发器（穿过去但能检测到）。

第六章：UI 系统

using UnityEngine;
using TMPro;

public class UIManager : MonoBehaviour
{
    public TextMeshProUGUI scoreText;
    public GameObject gameOverPanel;
    private int score = 0;

    public void AddScore(int points)
    {
        score += points;
        scoreText.text = "Score: " + score;
    }

    public void ShowGameOver()
    {
        gameOverPanel.SetActive(true);
        Time.timeScale = 0f;
    }
}

第八章：音频与粒子效果

public class SoundManager : MonoBehaviour
{
    public static SoundManager Instance;
    public AudioClip jumpSound;
    public AudioClip bgm;
    private AudioSource audioSource;

    void Awake()
    {
        Instance = this;
        audioSource = GetComponent<AudioSource>();
        audioSource.clip = bgm;
        audioSource.loop = true;
        audioSource.Play();
    }

    public void PlaySFX(AudioClip clip)
    {
        audioSource.PlayOneShot(clip);
    }
}
