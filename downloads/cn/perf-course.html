第一章：性能分析工具入门

性能优化的第一步是「测量」而非猜测。Unity Profiler 是最重要的工具。

Unity Profiler: Window → Analysis → Profiler (Ctrl+7)

关键指标：
├── CPU Usage    → 每帧 CPU 耗时
├── GPU Usage    → 渲染耗时
├── Memory       → 内存分配
├── Rendering    → Draw Call、三角面数
└── Physics      → 物理计算耗时

目标值（60fps）：
  每帧预算 = 1000ms / 60 = 16.67ms
  CPU < 12ms
  Draw Call < 100（手机）
  三角面 < 100K（手机）
💡 务必在目标设备上测试，编辑器中的性能数据不代表实机表现。
